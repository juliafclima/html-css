Principais formatações em HTLM5

“Semântica é o significado dos vocábulos, por oposição à sua forma.” - Michaelis

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a TAG B (não semântica - CSS).

Nesta, temos um termo em destaque usando a TAG STRONG (semântica - HTML).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a TAG I (não semântica - CSS).

Nesta, temos um termo em ênfase usando a TAG EM (enphase) - (semântica - HTML).

Texto marcado

Podemos criar também um texto marcado usando a TAG MARK.

Texto grande / Texto pequeno

Estamos criando um texto grande usando a TAG BIG e um texto pequeno usando a TAG SMALL nesse parágrafo.

Texto excluido

Podemos marcar um texto como excluido usando a TAG DEL para indicar que ele deve ser lido mas não considerado.

Texto sublinhado / inserido

Podemos marcar um texto sublinhado usando a TAG U (não semânica - HTML).

Existe também, o texto inserido usando a TAG INS (inserted) para dar uma ênfase e indicar que ele foi adicionado depois (semântica - CSS).

Texto sobrescrito

Para inserir coisas do tipo: X20 + 3 usamos a TAG SUP (superscript).

Texto subscrito

Para inserir coisas do tipo: H20 usamos a TAG SUB (subscript).
