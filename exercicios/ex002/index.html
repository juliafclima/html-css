Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito.
Basta colocar tudo no meio do par de tags <P> e </P>.

Se precisar quebrar o texto em algum lugar específico
esse, você pode usar a tag <BR>.

Vamos adicionar alguns simbolos especiais:

  • ®
  • ©
  • ™
  • €
  • £
  • ¥
  • ¢
  • Δ
  • ↑

Para isso acontecer, devemos usar a tabela HTML ENTITIES.

Vamos adicionar alguns emoji:

  • ✋
  • 🖖
  • 🤓
  • 🍟

Para adicionar um emoji, basta utilizar &#x + CÓDIGO começando pelo 1F.
