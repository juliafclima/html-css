Outras formatações em HTML5

Código-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript. A TAG mono-espaçada utilizada para fazer isso foi a CODE.

Pré-formatado

    num = int(input("Digite um número: "))
    if num % 2 == 0;
        print(f'O número {num} é PAR')
    else;
        print(f'O número {num} é ÍMPAR')
    print('Fim do programa!')

Para insirir um texto da exata forma que se foi escrito, pode-se combinar a TAG CODE com a TAG PRE (pré formatada).

Citações simples

Como diria o pai de um amigo: O computador é um burro muito rápido. Nesse paragrafo usamos a TAG Q (Q de quote) para fazer uma citação.
